# Maze generator sample application

## How to run

### To build docker image

```
docker-compose build
docker-compose run
```

### Run through console

Navigate to the solution and run
```
dotnet restore
dotnet build
dotnet run --project Maze.csproj
```
Then go to http://localhost:64497
